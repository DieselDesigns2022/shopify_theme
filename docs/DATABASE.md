# Data and Database Reference

## Local Database Status
This theme has no local database. Do not invent tables or local data models.

## Shopify-Managed Data
Shopify owns and manages commerce data.

## Products
Products are Shopify-managed and planned for later theme templates.

## Variants
Variants are Shopify-managed and planned for later product work.

## Inventory
Inventory is Shopify-managed and not stored in the theme.

## Collections
Collections are Shopify-managed and planned for later templates.

## Customers
Customers are Shopify-managed. Theme account integration is planned.

## Orders
Orders are Shopify-managed and not stored in the theme.

## Cart Data
Cart data is Shopify-managed. Cart templates and behavior are planned.

## Line-Item Properties
Line-item properties are planned for personalization where Shopify supports them.

## Metafields
Metafields are planned structured data sources and are not implemented now.

## Metaobjects
Metaobjects are planned reusable structured content sources and are not implemented now.

## Theme Settings
Theme settings are stored by Shopify and represented by `settings_schema.json` and `settings_data.json`.

## External Services
External services are not currently connected.

## Future Documentation Rules
Document only real Shopify-managed data usage when features are implemented.
