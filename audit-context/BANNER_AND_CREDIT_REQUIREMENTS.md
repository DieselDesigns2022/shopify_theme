# Required Custom Banner System

The theme must include flexible banner sections that allow the store owner or Diesel Designs to upload custom-designed graphics whenever needed.

## Welcome Banner

A welcome banner is required.

It can be placed:

- Directly below the header
- Below the homepage hero
- Between homepage sections
- On selected page templates

Settings should include:

- Desktop banner image
- Separate mobile banner image
- Optional clickable link
- Optional button
- Image width
- Maximum height
- Full-width or contained layout
- Rounded or square corners
- Top and bottom spacing
- Mobile visibility
- Desktop visibility
- Optional alt text

The banner should support a fully designed image without forcing text, buttons, or overlays on top of it.

## Custom Graphic Banner

A reusable custom banner section should be available throughout the theme.

It can be used for:

- Sales
- New releases
- Store announcements
- Shipping notices
- Seasonal promotions
- Collection promotions
- Rewards programs
- Custom orders
- Wholesale information
- Upcoming events
- Email signup promotions

The store owner can upload a complete custom graphic and optionally make the entire banner clickable.

## Divider Banners

The theme must include divider banners that can be placed between sections.

Divider banner options should include:

- Full-width image divider
- Contained image divider
- Thin decorative banner
- Tall promotional divider
- Desktop image
- Separate mobile image
- Optional clickable link
- Adjustable spacing
- Adjustable height
- Image fit controls
- Optional repeating image
- Optional background color

Divider banners should work between any homepage section and on flexible page templates.

## Collection Banners

Each collection should support:

- Main collection banner
- Separate mobile banner
- Optional banner above the product grid
- Optional banner inside the product grid
- Optional banner below the product grid
- Clickable promotional banner
- Seasonal banner override

This allows shops to promote related products, sales, launches, or important information while customers browse a collection.

## Product Page Banners

Product pages should allow optional custom banners:

- Above the product information
- Below the Add to Cart area
- Between product-page sections
- Above related products
- Near shipping or processing information

Possible uses include:

- Turnaround-time notices
- Custom-order reminders
- Shipping promotions
- Product-care graphics
- Size-chart graphics
- Sale announcements

## Page and Blog Banners

Flexible banner sections should also be available on:

- Standard pages
- About pages
- FAQ pages
- Contact pages
- Blog pages
- Blog articles
- Landing pages

## Banner Design Requirements

All banner sections must:

- Work with graphics created outside Shopify
- Keep uploaded images sharp and responsive
- Support separate desktop and mobile graphics
- Allow the banner to remain image-only
- Avoid automatically cropping important artwork
- Include accessible alt-text settings
- Support optional links
- Allow spacing and width adjustments
- Be rearrangeable through the Theme Editor
- Avoid adding forced text or design elements over custom graphics

# Permanent Diesel Designs Credit

The footer credit is always required.

Every installation must display:

**Website by Diesel Designs**

“Diesel Designs” must link to:

**https://www.dieseldesigns.co**

There will be:

- No Theme Editor setting to disable it
- No credit-removal license
- No higher-priced version without the credit
- No alternate footer layout without the credit

The requirement will also be stated clearly in:

- The theme license
- Product listing
- Customer documentation
- Installation instructions
- Support terms
